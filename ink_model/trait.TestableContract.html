Deploys the contract given the provided arguments for deployment.

Note

This shall be performed only once during the lifetime of a contract.

Panics

This might panic if the provided arguments do not match the expected.

Calls the contract with the given message and its inputs and upon successful execution returns its result.

Note

Takes &mut self since it could potentially call a message that mutates state. There currently is no separation between messages that mutate state and those that do not.

Panics

If the contract has no message handler setup for the given message.
